# Progettazione dell'intersuola di una scarpa da running

## Overview
L’evoluzione delle calzature sportive rappresenta uno degli sviluppi ingegneristici 
più significativi nell’ambito dello sport degli ultimi anni; le scarpe da running sono
ormai veri e propri sistemi complessi progettati al fine di garantire comfort, maggiore
efficienza energetica e le migliori prestazioni possibili a chi le indossa.
Nel corso degli ultimi decenni si è sempre più affermato l’impiego di piastre in
fibre di carbonio all’interno dell’intersuola, modificando in maniera significativa il
comportamento meccanico delle scarpe e le prestazioni che quest’ultime potevano
garantire.

Al fine di comprendere a fondo l’importanza di questo studio, è necessario analizzare
il principio meccanico alla base delle scarpe da running rinforzate con piastre
in carbonio e la loro evoluzione storica.
